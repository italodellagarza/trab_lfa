Este programa minimiza um autômato finito determinístico completo, no 
formato descrito no arquivo-texto exemplo, gerando o autômato resultante 
e a tabela de minimização.

Uso:

python3 minimiza.py <automato-de-entrada>.txt <tabela-de-saida>.txt 
<automato-de-saida>.txt

O programa foi feito em Python 3.6.1 e testado usando GNU/Linux e 
Windows 10.
